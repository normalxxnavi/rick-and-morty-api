Diseñado por: Javier López Velasco

Hola Soy Javier López Velasco, estudiante de 2ºDAW para la asignatura Desarrollo en Entorno Cliente. Mi proyecto se centra en una API de Rick y Morty donde recupero toda la informacion de todos sus personajes y localizaciones de la serie.

Este proyecto esta desarrollado en HTML5, CSS, y JS

La utilidad de JS es que utilizo una funcion fetch() para recuperar de informacion a traves del protocolo HTTP o HTTPS, donde fetch() tiene una promesa que te devuelve informacion
